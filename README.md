# Glend-Skill

Glend-Skill is an AI agent skill that enables agents to interact with the GemachDAO Glend-Front-End protocol. It adheres to the skills.sh specification.

## Overview

The GemachDAO Glend-Front-End is a Lend and Borrow Platform (DeFi). This skill package provides the necessary context and instructions for AI agents to help build, interact with, and understand the protocol.

## Installation

```bash
npx skills add GemachDAO/glend-skill
```

## How it works

This repository acts as an agent protocol instruction set:
- `SKILL.md`: Contains the core instructions for the AI agent, explaining how to interact with the project.
- `package.json`: Contains metadata conforming to the skills.sh standard.
